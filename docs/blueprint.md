# **App Name**: Lorekeeper Admin

## Core Features:

- Auth Gateway: Secure email and password authentication system using the standalone Supabase JS client for session management.
- Protected Admin Shell: A layout-protected dashboard environment featuring a sidebar with placeholder navigation for future modules.
- Supabase Client Core: Standardized configuration of @supabase/supabase-js to handle direct database and auth interactions.
- Component Library Integration: Unified UI foundation using shadcn/ui primitives such as buttons, cards, and inputs for consistent admin design.

## Style Guidelines:

- Primary: Regal Gold (#E6B81A) for buttons, accents, and high-importance UI elements.
- Background: Deep Obsidian (#171512), a dark desaturated theme for professional administrative workflows.
- Headlines and Labels: 'Space Grotesk' to provide a modern, structural digital-archive aesthetic.
- Body and Tables: 'Inter' for high legibility and neutral presentation of administrative data.
- Sidebar-first architecture with a fixed navigation rail and a responsive scrollable content viewport.
- Minimalist, thin-stroke icons to signify navigation items like Cards, Events, and Players without visual clutter.
